Steps to install:

1. Install Django and Python: https://docs.djangoproject.com/en/dev/topics/install/

2. Checkout the source code

3. In WebApp.settings, change the DATABASES hash table NAME field to the path desired

4. run

manage.py syncdb 

in your source code folder

5. run 

manage.py runserver
